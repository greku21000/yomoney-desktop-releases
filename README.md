# YOMONEY Desktop

System-wide gambling site blocker for Windows.

**Download the latest installer:**
👉 [Releases page](https://github.com/greku21000/yomoney-desktop-releases/releases/latest)

## What it does

- Blocks gambling sites at the DNS layer — works in **every browser and every app**, not just Chrome
- Tamper-protected: a **24-hour cooldown plus trusted-person veto** is required to disable
- Self-installs a local certificate so blocked HTTPS sites show a YOMONEY-branded page instead of a browser error
- Requires an active Pro subscription on [yomoney.tech](https://yomoney.tech)

## System requirements

- Windows 10 or 11, 64-bit
- ~50 MB free disk space
- Admin rights (the app requests UAC on first launch to write the system hosts file)
